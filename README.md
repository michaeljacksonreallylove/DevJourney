# README

# DevJourney（デブジャーニー）

プログラミング学習者のための「情報共有・学習コミュニティ」アプリケーション

---

## 1. コンセプト

私自身も一人のプログラミング学習者として、学び続ける上で最も難しいのは「モチベーションの維持」だと実感しています。
わからないことをすぐ誰かに相談できたり、一緒に勉強する仲間の存在を身近に感じられれば、学習はずっと続けやすくなります。
DevJourney はそのための場です。教材の共有や学習グループ、もくもく会を通じて、「学び続ける習慣」を支援します。

---

## 2. 目的

- 他の学習者との教材やノウハウの共有
- オンライン学習スペース（Zoom / Google Meet 等）による学習ハードルの軽減
- 学習コミュニティの作成・参加を通じたモチベーション維持

---

## 3. 想定ユーザー

- 自習・転職準備・資格取得などを目的とするプログラミング学習者
- 情報収集やコミュニティ内で学習を進めたい学習者

---

## 4. 主要機能

### 1) ログイン・プロフィール
- ユーザー名、自己紹介、SNSリンク（GitHub 等）登録
- プロフィール画像（avatar）登録・編集
- 他ユーザーのプロフィール閲覧、フォロー / DM（将来的に）

### 2) 教材おすすめ投稿(実装中)
- 書籍、動画、オンライン講座、技術記事などの投稿
- タイトル、URL、説明、カテゴリを登録可能
- コメントや「いいね」でリアクション
- OpenAI API を使った教材推薦（キーワードに応じた関連投稿表示）

### 3) いつでも参加できるオンラインもくもく会（実装予定）
- ユーザーがいつでも開催・参加できるもくもく会機能

### 4) 学習グループ（実装予定）
- グループの作成・参加
- グループごとのチャットルーム

---

## 5. 技術スタック
- バックエンド：Ruby on Rails 8
- フロントエンド：HTML / CSS / JavaScript（必要に応じて Bootstrap）/Figma
- データベース：PostgreSQL
- 管理：GitHub

---

## 6. セットアップ（開発環境）

1. リポジトリをクローン

```bash
git clone https://github.com/michaeljacksonreallylove/DevJourney.git
cd DevJourney
```

2. 依存関係をインストール

```bash
bundle install
yarn install
```

3. データベース作成・マイグレート

```bash
bin/rails db:create db:migrate 
```

4. サーバ起動

```bash
bin/rails server
```

※ 画像処理（CarrierWave + MiniMagick 等）を使う場合は ImageMagick のインストールが必要です（macOS では `brew install imagemagick` 等）。
